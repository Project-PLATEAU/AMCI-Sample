# AMCI Sample

このファイルはPLATEAUの利用促進の為、https://amci.tokyo-omy-w.jp/ で公開されているデータを元に作られた学習用のサンプルデータです。

Nuxt.jsにて開発を行っております。tNuxt.js
https://nuxtjs.org/

## Setup

```bash
# 依存ファイルのインストール
$ npm install

# localhost:3000 でローカルサーバを起動
$ npm dev
```

# WebGL関連のファイル

- /src/components/gl/js/: こちらのフォルダ以下にPLATEAUを利用したWebGLの実装データが入っています
